Financas

O programa:

O programa Finanças tem o objetivo de poder cadastrar ganhos e gastos, divididos em categorias. Com esses dados cadastrados é possível gerar relatórios básicos.

Com esses relatórios é possível ter uma melhor noção de onde vem e para onde vai o seu dinheiro em cada mês, ano ou em todo o período de dados. Também é possível saber qual o impacto de cada categoria no orçamento, por exemplo ao saber qual a porcentagem de cada categoria de ganhos na renda total e e também com quais categorias se tem os maiores gastos.

Para rodar o Programa:

Para rodar o projeto na Eclipse IDE é preciso importar o JavaFX. Instruções sobre como importar as dependências de forma correta podem ser encontradas nesse [video tutorial](https://www.youtube.com/watch?v=wkuKISa3IHI)


Compilar:

Usando o eclipse, é possível compilar o projeto para gerar um jar executável no menu File -> Export e escolher a opção Runnable JAR file. Após compilar, é ṕossível executar o programa com "java -jar <programa>.java"
